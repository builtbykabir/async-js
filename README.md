# Async JavaScript Examples

This repository contains code examples from my personal practice with asynchronous JavaScript concepts. Each folder corresponds to a specific topic, and the code was created to reinforce my understanding of async patterns and behaviors in JavaScript.

## Topics Covered (alphabetical order)

- [**async-await**](./async-await) — Using async functions and await expressions

- [**block-main-thread**](./block-main-thread) — Demonstrating how synchronous code can block the browser

- [**callback-hell**](./callback-hell) — Nested callbacks and their pitfalls

- [**dog-api-fetch**](./dog-api-fetch) — Fetching data from the Dog API using fetch

- [**dog-api-xhr**](./dog-api-xhr) — Fetching data from the Dog API using XMLHttpRequest

- [**fetch-api**](./fetch-api) — Basics of the Fetch API

- [**fix-callback-hell**](./fix-callback-hell) — Refactoring callback-heavy code into cleaner patterns

- [**promises**](./promises) — Creating and handling Promises in JavaScript

- [**try-catch**](./try-catch) — Error handling in asynchronous code
